Syntax Description
Search commands
\x Hexadecimal (e.g. \x09)
\\ For \ (backslash)
\. Any sign
\w Any word
\# Any number
\t Tab (0x09)
\b Tab or blank (0x20, 0x09)
\r CR Carriage Return (0x0d)
\n LF Line Feed (0x0a)
\d 0-9 (digits)
\m A-Z, a-z, 0-9 (alphabetic characters or digits)
\a A-Z, a-z (alphabetic characters)
\s12 Skip number of characters (eg \s17 skip 17 characters)
\ixxx\i Move the cursor to the position after text 'xxx'
Value Syntax
\v Value (actually this needs to be a number. Text does not work)
\1 Byte 1 (lowest significant bit)
\2 Byte 2
\3 Byte 3
\4 Byte 4
\h Read hex value
